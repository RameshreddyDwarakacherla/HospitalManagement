# 🏥 Hospital Management System - HealthMate Project

This is a **Java-based Hospital Management System** developed as a full-stack academic project.

> 🔧 **Cloned and personalized by [Rameshreddy Dwarakacherla](https://github.com/RameshreddyDwarakacherla)**  
> 📁 GitHub Repo: [HospitalManagement](https://github.com/RameshreddyDwarakacherla/HospitalManagement)

---

## 📌 Features

- Admin can log in with default credentials: `admin / admin`
- Admin functionalities:
  - Add new patients
  - Update patient info
  - Search and discharge patients
- Doctor functionalities:
  - Register via form
  - Login to see assigned patients
  - Export patient list as `.doc` file
